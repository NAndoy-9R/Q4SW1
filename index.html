Math Methods in JavaScript

JS Math Method - Easy

Determine the value of roundedPrice

var sandwichPrice = 125.80;
var roundedPrice = Math.round(sandwichPrice);

Explanation: Math.round rounds the number to the nearest integer.
sandwichPrice = 125.80. Rounding 125.80 gives 126.

Return Value: 126

Determine the value of standardSandwiches

var miniSandwiches = 81;
var standardSandwiches = Math.sqrt(miniSandwiches);

Explanation: Math.sqrt calculates the square root of a number.
miniSandwiches = 81. The square root of 81 is 9.

Return Value: 9

Determine the value of absoluteDifference

var priceDifference = -15.50;
var absoluteDifference = Math.abs(priceDifference);

Explanation: Math.abs returns the absolute value of the number (removes the negative sign).
priceDifference = -15.50, so the absolute value is 15.50.

Return Value: 15.50

Determine the value of totalCost

var basePrice = 150;
var totalCost = basePrice ** 2;

Explanation: The '**' operator is used for exponentiation in JavaScript.
basePrice = 150. 150 raised to the power of 2 (150 ** 2) equals 22500.

Return Value: 22500

Determine the value of maxSales

var dailySales = [2500, 3200, 2800, 3500];
var maxSales = Math.max(...dailySales);

Explanation: Math.max returns the largest number in a set of arguments. The spread operator (...) is used to pass the array elements as individual arguments.
The maximum sales value from the array [2500, 3200, 2800, 3500] is 3500.

Return Value: 3500

Determine the value of roundedPrice after a discount

var sandwichPrice = 135.75;
var discountedPrice = sandwichPrice - (sandwichPrice * 0.15);
var roundedPrice = Math.round(discountedPrice);

Explanation: First, we calculate the discounted price. A 15% discount is applied to the original price (135.75 * 0.15 = 20.3625).
Subtracting the discount from the original price gives: 135.75 - 20.3625 = 115.3875. Rounding this to the nearest integer results in 115.

Return Value: 115

Determine the value of familyPack

var miniSandwiches = 121;
var standardSandwiches = Math.sqrt(miniSandwiches);
var familyPack = Math.pow(standardSandwiches, 2);

Explanation: First, the square root of miniSandwiches (121) is calculated, which is 11.
Then, Math.pow raises the number 11 to the power of 2, which results in 121.

Return Value: 121

Determine the value of finalAdjustment

var priceDifference = -22.50;
var absoluteDifference = Math.abs(priceDifference);
var finalAdjustment = absoluteDifference * 2;

Explanation: Math.abs returns the absolute value of priceDifference, which is 22.50.
Doubling this value gives 45.00.

Return Value: 45.00

Determine the value of roundedTotal

var basePrice = 160;
var taxRate = 0.12;
var totalPrice = basePrice + (basePrice * taxRate);
var roundedTotal = Math.round(totalPrice * 100) / 100;

Explanation: First, calculate the total price by adding the tax: 160 + (160 * 0.12) = 160 + 19.2 = 179.2.
Then, rounding 179.2 to two decimal places gives 179.2.

Return Value: 179.2

Determine the value of averageSales

var dailySales = [2750, 3100, 2950, 3300, 3050];
var minSales = Math.min(...dailySales);
var maxSales = Math.max(...dailySales);
var averageSales = (minSales + maxSales) / 2;

Explanation: First, Math.min and Math.max return the minimum (2750) and maximum (3300) values from the array.
The average sales value is then calculated by averaging these two numbers: (2750 + 3300) / 2 = 6050 / 2 = 3025.

Return Value: 3025
